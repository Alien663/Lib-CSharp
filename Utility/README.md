# Alien.Common.Utility

General-purpose C# extensions for string, date, and object helpers.

## 📦 Installation

```bash
Install-Package Alien.Common.Utility
```

## 🚀 Features
- String manipulation
- Date formatting
- Object cloning & conversion

## 🧪 Example Usage

```csharp
using Alien.Common.Utility;

string s = " 全形字 ＡＢＣ ";
string half = s.ToHalfWidth();
```

## 📘 Dependencies
- None

## 🛠 Compatibility
- .NET 6 and above
